3D Printing

For this project, we used the block art we made in Minecraft and transformed it into 3D printed models. Mineways software was used to extract the block data and turn it into an STL file, which was then used for each of the prints.

1st Print

This is my first print I did of Gudetama. No eyes are showing do the STL file not rendering the color. The base was kept to represent the egg white Gudetama sits on.

2nd Print (with updated model)

This is my second print of Gudetama. Although not physically larger in scale, the model is based on a larger block model made in Minecraft, so the model is a bit more detailed. Supports can be seen under the arms and inside the eyes, although the small size of the model makes them difficult to remove.

Final Print

This is the final print of Gudetama, based on the same model as the 2nd print, made in the Makerspace Lab. Issues with not printing in the correct size resulted in a merging of the support and front body, resulting in a loss of detail.
